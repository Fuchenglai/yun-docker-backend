### 项目背景
该项目脱胎于华南农业大学的虚拟云实验平台，原平台用于提供实验环境给本校学生，学生可以通过老师设计的镜像来定制化创建容器，解决传统物理机部署实验环境消耗资源多配置复杂的问题。
我在参与原项目的开发开发工作后，受到启发开发了云托管平台，前后端代码的开发工作都是个人完成。部分技术栈与原平台相似，也引入了支付宝沙箱，websocket等新技术。
### 项目功能
帮助开发者通过该平台快速部署自己的web项目，解决了部署小型项目需要购买、配置服务器等痛点。用户可将自己的项目打包成镜像，然后在该平台拉取镜像并且创建容器，创建容器会消耗用户的积分，当积分消耗完可在该平台进行充值。
主要功能包括：拉取/查看/删除镜像，创建/重启/停止/启动/删除/查看容器，查看容器日志，查看容器实时信息参数,积分充值，订单管理，支付宝沙箱支付。
